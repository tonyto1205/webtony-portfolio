Sticker Game

A system for saving sticker images and uploading them as addressable assets to Bunny, exploring different approaches to building a flexible sticker collection system.

Overview

Sticker Game focuses on backend and content pipeline design rather than pure gameplay. The project experiments with how users create, manage, and sync sticker assets, using addressable assets hosted on Bunny to keep downloads small and content easily expandable.

Demo Video

The demo showcases creating sticker collections, saving images, and how new stickers can be delivered without full app updates.
