Home AI - Room Design

An AI-powered room and home redesign app. Uses Firebase to store notification data and API keys, then sends user images to Gemini AI to generate redesigned interiors and exteriors.

Overview

Home AI lets users capture or upload room photos, then applies AI-powered transformations to explore different interior and exterior styles. Firebase is used as a backend to manage notification settings, API keys, and user-related metadata.
