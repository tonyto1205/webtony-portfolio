Logo AI

An AI logo generation app using Firebase as backend to store notification data and API keys. Users send prompts to Alibaba Cloud AI to generate final logo images.

Overview

Logo AI streamlines the process of creating custom logos from simple text prompts. Firebase manages configuration, API keys, and notification data, while Alibaba Cloud AI generates logo concepts that users can pick from and refine.

Demo Video

The demo video shows how users write prompts, submit them to the AI backend, and browse through multiple generated logo options.
