Dream Decor: Home Makeover

A 3D home makeover game. Modified 3D models in Blender, implemented an inventory/backpack system and a PlayFab backend, plus cloud scripts for matchmaking and missions. Level scenes use cameras, effects, and lighting stored as addressables and uploaded to Bunny.

Overview

Dream Decor focuses on creating satisfying before/after makeovers. I handled environment preparation in Blender, set up inventory and item placement systems in Unity, and connected the game to PlayFab for player data, matchmaking, and mission progression. Lighting and camera setups are packaged as addressables to keep scenes flexible and easy to update.
