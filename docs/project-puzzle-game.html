Puzzle Game

A casual puzzle game where players connect the last three identical birds, each with different effects and behaviors.

Overview

This school project focuses on designing satisfying chain reactions and visual feedback as players connect matching birds. Different bird types trigger different effects, encouraging planning and experimentation.

Demo Video

The demo video shows core gameplay, bird effects, and how levels escalate in difficulty.
