Mobile Games

Commercial mobile games built with Unity, integrating live backend services, analytics, and monetization.

Crush Island: Choices & Love

A reality TV-style romance game. Stories are authored in Google Sheets, exported as CSV, converted to JSON, and served via Bunny CDN together with Spine characters. Includes Flowise-powered chat rooms plus Firebase, Facebook, and AdMob SDK integration.

Dream Decor: Home Makeover

A 3D home makeover game. Modified 3D models in Blender, implemented an inventory/backpack system and a PlayFab backend, plus cloud scripts for matchmaking and missions. Level scenes use cameras, effects, and lighting stored as addressables and uploaded to Bunny.

Fight Game

A combat-focused prototype where I created special effects and one-hit kill animations to emphasize powerful attacks.

Sticker Game

A system for saving sticker images and uploading them as addressable assets to Bunny, exploring different approaches to building a flexible sticker collection system.

Moonlight Ascension - Werewolf

A werewolf romance visual novel. Story data is managed in Google Sheets, exported as CSV, converted to JSON, and hosted with Spine characters on Bunny. Chat rooms use Flowise AI. Implemented upgrade card and Spine-based gacha systems, with data stored on PlayFab and Bunny.
