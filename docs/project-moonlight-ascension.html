Moonlight Ascension - Werewolf

A werewolf romance visual novel. Story data is managed in Google Sheets, exported as CSV, converted to JSON, and hosted with Spine characters on Bunny. Chat rooms use Flowise AI. Implemented upgrade card and Spine-based gacha systems, with data stored on PlayFab and Bunny.

Overview

Moonlight Ascension extends the narrative pipeline used in Crush Island with additional systems: upgradeable cards, Spine-based gacha animations, and progression data handled through PlayFab. Flowise-powered chat rooms add interactive conversations outside the main story flow.

The demo video features key romance scenes, gacha pulls, and examples of the Flowise chat integration.
